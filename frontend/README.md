# Frontend

UI-related code for the Takshashila 26 website.

- **`components/`** – React components (Navigation, Footer, EventCard, etc.)
- **`contexts/`** – React context providers (AuthContext, etc.)
- **`hooks/`** – Custom hooks (useGSAP, useLenis, etc.)

Imports use the `@/components`, `@/contexts`, and `@/hooks` path aliases (configured in `tsconfig.json`).
